# 🖌️ AI Comic Creator

Generate AI-powered comic panels in seconds.
Turn your ideas into comic-style visuals with ease.

---

## 🚀 Features

* AI-driven comic creation
* Multiple art styles
* Export as PNG or JPEG
* Easy-to-use interface

---

## 💻 Installation

```bash
git clone https://github.com/yourusername/ai-comic-creator.git
cd ai-comic-creator
pip install -r requirements.txt
python src/main.py
```

> For Node.js / web version, replace with `npm install` & `npm start`

---

## 🎨 Usage

1. Open the app
2. Enter your comic idea
3. Choose art style
4. Generate panels
5. Save / export
